Ce dépôt regroupe mes solutions pour le cours **CS50 Introduction to Computer Science** de Harvard (édition 2026).

## 🚀 Structure du Projet

| Semaine | Thème | Exercices Clés |
| :--- | :--- | :--- |
| **01** | C | Mario, Cash, Credit |
| **02** | Arrays | Readability, Caesar, Substitution |
| **03** | Algorithms | Runoff, Tideman |
| **04** | Memory | Volume, Filter, Recover |
| **05** | Data Structures | Speller, Inheritance |
| **06** | Python | DNA, Sentimental Problems |

## 🛠️ Utilisation Rapide

Le projet utilise un **Makefile** racine pour les exercices en C :
```bash
make "Week 1 — C/world/hello"
./"Week 1 — C/world/hello"
```

Pour les exercices en Python (**Week 6**) :
```bash
cd "Week 6 — Python/dna"
python3 dna.py databases/small.csv sequences/1.txt
```
